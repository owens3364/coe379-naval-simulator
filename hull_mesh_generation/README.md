# Hull Mesh Generation

## Setup

Run `uv sync` and activate the environment with `source .venv/bin/activate`. Depending on your platform, the `capytaine` install may not work correctly (see [here](https://capytaine.org/stable/developer_manual/installation.html#installation-for-development-using-pip-on-linux-or-wsl) for why); please try using `uv run --with capytaine python path/to/script.py`, or `uv pip install capytaine` and then `uv run python path/to/script.py`, to fix any installation issues if they occur. (This code was tested on an M1 Mac.)
